BELLEZA WEB
¿En qué consiste?

Belleza Web, como se puede deducir de su nombre, es una pequeña y sofisticada página web creada con la intención de que las mujeres tengan un lugar dónde encuentren todos los tipos de servicios que necesiten enfocados en la belleza. Tenemos los tres más importantes, que son MakeUp, Cabello y Manicure. Podrán encontrar una lista de cada uno de ellos, dónde y quién los realiza para que tengan mayor información y se podrán contactar con el profesional.

BELLEZA WEB
¿Cómo se creó?

Este sitio parte de un proyecto realizado para una asignatura,por tres chicas estudiantes de 3°año, con el objetivo de crear algo que fuera creativo, novedoso y útil. Así en conjunto , salió al aire esta idea,un buscador de belleza, que contenga información sobre distintos tipos de servicios de belleza en la ciudad de Temuco, Un solo sitio en el cual podemos comparar precios, ver que se ofrece para cada oficio y contactar.
